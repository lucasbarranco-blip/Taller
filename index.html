Avanzado (opcional)

¿Cómo consigo la configuración de Firebase? (una sola vez)
  1. Entrá a console.firebase.google.com con tu cuenta de Google y creá un proyecto (es gratis).
  2. Menú izquierdo → Firestore Database → Crear base de datos → modo producción → ubicación por defecto.
  3. Engranaje ⚙ (arriba a la izquierda) → Configuración del proyecto → sección "Tus apps" → botón Web </> → registrá la app.
  4. Firebase muestra un bloque firebaseConfig = { … }. Copiá lo que está entre las llaves y pegalo arriba.
  5. Volvé a Firestore → pestaña Reglas y publicá una regla que permita el acceso a la colección "talleres" (te la paso aparte).
